# User Module (Auth & Caching)

Thư mục này chịu trách nhiệm phục vụ dữ liệu trực tiếp lên Frontend bao gồm:

1. **Authentication:** 
    - Gắn kết NextAuth v5 để duy trì Session (OAuth: Google/Github).

2. **User Service:**
    - Insert kết quả thành công tìm được sang màn Search History Dashboard.

3. **Analytics Tracking:**
    - Lưu lượt tính của một nút Node (Wikipedia PagedID) nhằm build bản đồ hiển thị HeatMap các node có độ thông qua (Bắc Cầu) cao nhất.
